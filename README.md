# 🚀 Mohammad Ahmad — 3D Portfolio

A visually immersive personal portfolio for **Mohammad Ahmad**, Backend Developer & Full Stack Engineer. Built with cutting-edge web technologies to deliver an interactive 3D experience directly in the browser.

🔗 **Live Site:** [3d-portfolio-orpin-one.vercel.app](https://3d-portfolio-orpin-one.vercel.app)
📁 **Repository:** [github.com/Bitsnbytes14/3d-portfolio](https://github.com/Bitsnbytes14/3d-portfolio)

---

## ✨ Features

- **Interactive 3D Scenes** — Animated 3D models and geometries powered by Three.js and React Three Fiber
- **Smooth Animations** — Scroll-driven and hover-triggered transitions using GSAP
- **Responsive Design** — Fully optimized across desktop, tablet, and mobile
- **Fast Performance** — Built on Vite for lightning-fast dev server and optimized production builds
- **3D Skills Section** — Skills showcased through interactive 3D objects
- **Work & Projects Section** — Highlighted projects with descriptions and links
- **Experience Timeline** — Animated career history section

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18+ and npm

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/Bitsnbytes14/3d-portfolio.git

# 2. Navigate into the project
cd 3d-portfolio

# 3. Install dependencies
npm install
```

### Run Locally

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```

## 🎨 Customization

To personalize this portfolio with your own content, edit the files inside:

```
src/data/
```

This folder contains all the data powering the site — projects, skills, work experience, and more. No changes to components are needed for a basic rebrand.

## ☁️ Deployment

The project is deployed on **Vercel**. To deploy your own fork:

1. Push your repository to GitHub
2. Import it at [vercel.com/new](https://vercel.com/new)
3. Click **Deploy**
